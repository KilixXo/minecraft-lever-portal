# LeverPortal - Плагин для Minecraft

Плагин для Minecraft версии 1.21.11+, который позволяет игрокам создавать связанные порталы, активируемые рычагами, с системой оплаты алмазами.

## Возможности

- **Создание порталов из любых блоков**: Стройте порталы из любых блоков в любой форме
- **Вертикальная или горизонтальная ориентация**: Выбирайте, как будет ориентирован ваш портал
- **Активация рычагом**: Используйте рычаги для активации/деактивации порталов
- **Двусторонний проход**: Проходите через порталы в обоих направлениях
- **Множество пар порталов**: Создавайте столько пар порталов, сколько нужно
- **Экономическая система**: Оплата алмазами за создание и использование порталов

## Установка

1. Скачайте JAR-файл плагина
2. Поместите его в папку `plugins` вашего сервера
3. Перезапустите сервер
4. Плагин создаст папку `LeverPortal` в директории plugins

## Использование

### Создание портала

1. **Начните создание портала**
   ```
   /portal create <имя> <vertical|horizontal>
   ```
   Пример: `/portal create портал1 vertical`
   
   Система покажет стоимость создания портала в алмазах.

2. **Подтвердите создание**
   ```
   /portal confirm
   ```
   Оплатите стоимость создания алмазами из инвентаря.

3. **Выберите блоки портала**
   - Кликните правой кнопкой мыши по блокам, чтобы добавить их в портал
   - Можно использовать любые твёрдые блоки
   - Добавьте минимум 2 блока для формирования портала
   - Блоки могут быть в любой форме или узоре

4. **Завершите создание портала**
   ```
   /portal finish
   ```
   Или отмените: `/portal cancel`

5. **Повторите для второго портала**
   - Создайте ещё один портал с другим именем
   - Пример: `/portal create портал2 vertical`

### Связывание порталов

После создания двух порталов свяжите их вместе:
```
/portal link <портал1> <портал2>
```
Пример: `/portal link портал1 портал2`

**Требования для связывания:**
- Оба портала должны иметь одинаковую ориентацию (оба вертикальные или оба горизонтальные)
- Порталы должны находиться на расстоянии минимум 3 блока друг от друга
- Порталы должны быть в одном мире

### Активация порталов

1. **Установите рычаг**
   - Поместите рычаг в пределах 10 блоков от вашего портала
   - Кликните правой кнопкой по рычагу, чтобы связать его с порталом

2. **Переключите портал**
   - Переведите рычаг в положение ВКЛ, чтобы активировать оба связанных портала
   - Переведите рычаг в положение ВЫКЛ, чтобы деактивировать их

3. **Путешествуйте**
   - Войдите в активный портал, чтобы телепортироваться к связанному порталу
   - Работает в обоих направлениях!

### Управление порталами

**Список всех порталов:**
```
/portal list
```

**Удалить портал:**
```
/portal remove <имя>
```

## Экономическая система

### Стоимость создания порталов
Стоимость создания портала рассчитывается по формуле:
```
Стоимость = Базовая_стоимость + (Расстояние / Блоков_на_алмаз) * Стоимость_за_расстояние
```

По умолчанию:
- Базовая стоимость: 10 алмазов
- Блоков на алмаз: 100
- Стоимость за расстояние: 5 алмазов
- Максимальная стоимость: 1000 алмазов

### Стоимость телепортации
- **Глобальная стоимость**: 1 алмаз за телепортацию (по умолчанию)
- **Стоимость для портала**: Администратор может установить индивидуальную цену для каждого портала
- **Стоимость для игрока**: Администратор может установить индивидуальную цену для конкретного игрока на конкретном портале

Приоритет цен:
1. Индивидуальная цена игрока для портала
2. Цена портала
3. Глобальная цена

## Команды

### Для игроков

| Команда | Описание | Право |
|---------|----------|-------|
| `/portal` | Показать меню помощи | `leverportal.use` |
| `/portal create <имя> <ориентация>` | Начать создание портала | `leverportal.create` |
| `/portal confirm` | Подтвердить и оплатить создание | `leverportal.create` |
| `/portal finish` | Завершить создание портала | `leverportal.create` |
| `/portal cancel` | Отменить создание портала | `leverportal.create` |
| `/portal link <портал1> <портал2>` | Связать два портала | `leverportal.link` |
| `/portal remove <имя>` | Удалить портал | `leverportal.remove` |
| `/portal list` | Список всех порталов | `leverportal.use` |
| `/portal cost <портал>` | Узнать стоимость телепортации | `leverportal.use` |

### Для администраторов

| Команда | Описание | Право |
|---------|----------|-------|
| `/portal setcost <портал> <цена>` | Установить цену для портала | `leverportal.admin` |
| `/portal setplayercost <игрок> <портал> <цена>` | Установить цену для игрока | `leverportal.admin` |

## Права доступа

- `leverportal.use` - Базовое использование порталов (по умолчанию: op)
- `leverportal.create` - Создание порталов (по умолчанию: op)
- `leverportal.link` - Связывание порталов (по умолчанию: op)
- `leverportal.remove` - Удаление порталов (по умолчанию: op)
- `leverportal.admin` - Управление экономикой и ценами (по умолчанию: op)

## Примеры

### Пример 1: Простой вертикальный портал
```
/portal create дом vertical
/portal confirm  # Оплатить создание
[Кликните правой кнопкой по 4 блокам в вертикальную линию]
/portal finish

/portal create база vertical
/portal confirm  # Оплатить создание
[Кликните правой кнопкой по 4 блокам в вертикальную линию]
/portal finish

/portal link дом база
[Поместите рычаг рядом с порталом "дом" и переключите его]
```

### Пример 4: Настройка цен (для администраторов)
```
# Установить цену 5 алмазов для портала "дом"
/portal setcost дом 5

# Установить цену 2 алмаза для игрока Steve на портале "база"
/portal setplayercost Steve база 2

# Проверить стоимость телепортации
/portal cost дом
```

### Пример 2: Горизонтальная рамка портала
```
/portal create врата_незера horizontal
[Кликните правой кнопкой по блокам, формирующим горизонтальную рамку]
/portal finish

/portal create врата_обычного_мира horizontal
[Кликните правой кнопкой по блокам, формирующим горизонтальную рамку]
/portal finish

/portal link врата_незера врата_обычного_мира
[Поместите рычаг и активируйте]
```

### Пример 3: Портал произвольной формы
```
/portal create кастом1 vertical
[Кликните правой кнопкой по блокам в любом узоре - круг, квадрат и т.д.]
/portal finish

/portal create кастом2 vertical
[Кликните правой кнопкой по блокам в любом узоре]
/portal finish

/portal link кастом1 кастом2
```

## Технические детали

### Механика порталов
- Порталы могут быть сделаны из любых твёрдых блоков
- Форма портала полностью настраиваемая
- Минимум 2 блока требуется на портал
- Телепортация имеет задержку 2 секунды для предотвращения быстрой повторной телепортации
- Направление взгляда игрока (yaw/pitch) сохраняется при телепортации

### Хранение данных
- Данные порталов сохраняются в `plugins/LeverPortal/portals/portals.json`
- Данные автоматически сохраняются при выключении сервера
- Данные загружаются при запуске сервера

### Типы ориентации
- **Vertical (вертикальная)**: Портал стоит вертикально (как дверной проём)
- **Horizontal (горизонтальная)**: Портал лежит горизонтально (как портал в полу/потолке)

## Решение проблем

**Портал не активируется:**
- Убедитесь, что оба портала связаны
- Проверьте, что рычаг находится в пределах 10 блоков от портала
- Убедитесь, что рычаг правильно связан (должно появиться сообщение при установке)

**Не могу связать порталы:**
- Убедитесь, что оба портала имеют одинаковую ориентацию
- Проверьте, что порталы находятся на расстоянии минимум 3 блока друг от друга
- Убедитесь, что оба портала находятся в одном мире

**Телепортация не работает:**
- Убедитесь, что оба портала активированы (рычаг в положении ВКЛ)
- Проверьте, что вы проходите через блоки портала
- Подождите 2 секунды между телепортациями

## Совместимость версий

- **Версия Minecraft**: 1.21.11+
- **Серверное ПО**: Spigot, Paper или совместимые форки
- **Версия Java**: 17 или выше

## Компиляция

Для компиляции плагина используйте Maven:
```bash
mvn clean package
```

Скомпилированный JAR-файл будет находиться в папке `target/`.
